Demo(Basic)

"jQuery CompileHandlebarsTmpl.js" is a jQuery Plugin to use easily "Handlebars.js"

JSON

{
	"Persons": [
		{
			"firstName": "Sato",
			"lastName": "Daiki"
		},
		{
			"firstName": "Hoge",
			"lastName": "Fuga"
		}
	]
}

Template

{{#each Persons}}
<div>{{firstName}} - {{lastName}}</div>
{{/each}}

jQuery

$(function() {
	$('.js-compile').compilehandlebarstmpl({
		jsonURL: 'json/test.json',
		getDataType: 'json',
		tmplSrc: '#tmpl'
	});
});
